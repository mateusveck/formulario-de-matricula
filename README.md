# Formulário de Matrícula

Projeto de um formulário de matrícula para uma escola infantil, desenvolvido durante meus estudos.

## Sobre o projeto

A ideia foi criar uma página de formulário com campos para dados da criança, endereço, responsável, turno de estudo e opções de atividades.

Foi um projeto importante para praticar melhor a estrutura de formulários no HTML e a estilização dos campos com CSS.

## Tecnologias usadas

- HTML
- CSS
- Git
- GitHub

## O que pratiquei

- Formulários no HTML
- Inputs, selects, textarea e upload de arquivo
- Organização de campos por seções
- Estilização de formulário
- Uso de classes e variáveis no CSS
